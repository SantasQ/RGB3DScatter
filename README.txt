
RGB 3D Scatter Viewer for Tableau + GitHub Pages

1. このフォルダを GitHub にアップロード（例: RGB3DScatter）
2. GitHub Pages を有効化して、ブランチを main / docs などに設定
3. Tableau の Web ページオブジェクトに以下のように設定：
   https://yourusername.github.io/RGB3DScatter/index.html?image=The%20Scream
4. サムネイル画像をクリックした際に ImageName を URL パラメータとして渡すように設定

このページは ImageName（原題）で RGB 点群を自動的に切り替えて描画します。
